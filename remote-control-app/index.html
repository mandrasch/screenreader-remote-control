Screenreader remote control

Mac / Voice Over

Open a website

Activate / Deactivate CMD + F5

Start / stop reading VO + A / CTRL

Wait a few seconds after clicking this - after the first item is read, you can use read next / previous element below.

Read next / prev VO + → / ←

Focusable items (SHIFT +) TAB

Into group VO + SHIFT + ↓

Jump to top VO + FN + ←
